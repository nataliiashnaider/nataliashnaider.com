ЙОГАРЕТРИТ — готовий сайт

Для публікації завантажте на хостинг:
- index.html
- папку images
- robots.txt
- sitemap.xml

Перед запуском:
1. Замініть YOUR-DOMAIN.COM у robots.txt та sitemap.xml на реальний домен.
2. Перевірте посилання Instagram у index.html.
3. За потреби додайте точну назву локації та умови передоплати.

Усі фотографії на сторінці використані по одному разу. Візуальні дублікати з отриманих файлів були виключені.
